Challenges Challenge and Plan to Success

Our challenge is to create cutting edge tool that:

  • Provides a robust and easy to use platform to gather information about the project from students and allow client/supervisor to monitor this activity.
  • Suitable with the existing platform.
  • is functional with attractive UI

As a team our challenges are to:

  • collaborate with other team and the client.
  • distribute the work (technical and non technical) within the team.
  • managing timetable to finish the project on time.

In order to be successful with this project, we have to meet some challenging requirements. Our paramount target is to create good relationship with the client and make sure our platform is the result of their initial idea at the first place. We want to create a robust and functional platform with a smooth and attractive user interface as it would draw more user (target user is university students).
