Future Developments

These are some improvements that can be made to features in the future:

  • It will be helpful for the supervisor to be able to filter through all of the projects based on the status.
  • For the request for resources modal, the options can be changed/ added to in the future. A notification should be sent to the teaching assistant or client. A popup should appear, telling the user that their request has been sent.
  • Clicking on a "visiblity icon" could lead to that specific user's page.
  • In the future, the names should automatically appear when a user makes a comment since the users will be logged in. They could possibly reveive notifications when a post is made on the board.
  • We could have tasks in the table automatically become overdue if they haven't been completed by the duedate set. We could also have filtering features.
  • The current biweekly report on the index page could state when the report is due, and perhaps send notifications to students if a deadline is approaching.
  • Within the report, there could be an option to add meetings and individual contributions as blocks.
